# Clean Work – Professional Cleaning Service Booking Platform

Clean Work is a modern, responsive single-page web application built with **React** that allows users to easily browse and book specialized cleaning services: Office Cleaning, Kitchen Cleaning, Car Washing, and Factory Cleaning.

### Main Features
- Clean homepage with service showcase
- Interactive service cards with descriptions
- Dynamic multi-step booking forms tailored to each service
- Date & time selection with past-date restriction
- Booking confirmation page with clear summary
- User-specific "Booking Progress" tracker (stored in localStorage)
- Responsive design (mobile-first, desktop-friendly)
- Modern blue-themed UI with smooth hover effects & animations

### Tech Stack
- React (hooks + context)
- React Router v6 (navigation & data passing via state)
- Context API (user authentication & login state)
- Vanilla CSS (modular styles + responsive breakpoints)
- Font Awesome icons
- localStorage (client-side booking persistence)
